# Simple Quiz App

[![Kotlin](https://img.shields.io/badge/Kotlin-1.8+-blue.svg?logo=kotlin)](https://kotlinlang.org)
[![Platform](https://img.shields.io/badge/Platform-Android-green.svg?logo=android)](https://www.android.com/)
[![Quiz](https://img.shields.io/badge/Type-Quiz_App-brightgreen.svg)]()

Простое приложение-викторина для изучения базовой логики Android приложений. Демонстрирует навигацию между фактами и проверку ответов.

## Функциональность

- Навигация между фактами (вперед/назад)
- Проверка правильности ответов (True/False)
- Toast-уведомления о правильности ответа
- Простая модель данных для фактов

## Технологии

- Kotlin
- Basic Android UI
- Activity navigation
- Simple data class model
